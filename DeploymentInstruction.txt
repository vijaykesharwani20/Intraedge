1. First Clone the repository using below command whichever directory you want
	git clone https://github.com/vijaykesharwani20/Intraedge.git
	
2. make sure you have apache tomcat insatalled in your machine

3. run command under the /intraedge folder
	mvn clean install
	
4. If you are using eclipse or intellij then click on project and click run as on server

5. select tomcat server which you have in you project and apply , It will run you project on server.

use postman to request our application. --> http://localhost:8080/Intraedge/colours
